# Weather App

This is a weather forecasting application, using React.

![weather-app](https://i.imgur.com/XIhXEzH.png)

**Built Using:**

- React.js
- Node.js

**Running the app**

- Clone the project
- Install Dependencies
- Move into directory and type: `npm install`
- From the directory and type: `npm start`
